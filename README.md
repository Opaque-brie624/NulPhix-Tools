## 🧾 Basic Troubleshooting

### The app does not open
- Make sure you extracted the ZIP file first
- Check that the file finished downloading
- Try running it as admin

### Windows blocks the file
- Open the file properties
- Check if Windows marked it as blocked
- If you trust the source, allow the file and try again

### The app closes right away
- Run it from a folder with full access, like Downloads or Desktop
- Check if your antivirus moved a file
- Download the latest release again

### No results appear
- Check the target you typed
- Make sure the device or host is online
- Try a different search term or port range
